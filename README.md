# Algorithm Visualizer

A professional algorithm visualization web application built with Go backend and Svelte frontend.

## Features
- Interactive algorithm visualization
- Real-time step-by-step execution
- Light/dark theme support
- Modern, responsive UI

## Tech Stack
- Backend: Go 1.21+
- Frontend: Svelte 5 + TypeScript
- Build: Vite
- Testing: Go testing + Vitest
